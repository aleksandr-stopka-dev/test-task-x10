# Интеграция каталога курсов (Тестовое задание)

Легковесный каталог онлайн-курсов с динамическим рендерингом данных, разработанный в качестве тестового задания для команды 10X.

## Стек технологий
* **Разметка:** HTML5 (Семантичная и доступная)
* **Стили:** SCSS (Модульная компонентная архитектура по методологии БЭМ)
* **Логика:** Vanilla JavaScript (ES6+, выполнение на стороне клиента без фреймворков и библиотек)

## Ключевые особенности и оптимизация

* **Резиновый адаптив (Без жестких брейкпоинтов):** Интерфейс плавно масштабируется от 1920px до 320px. Используются CSS-функции `clamp()` для попиксельного изменения размеров и `CSS Grid` с параметром `repeat(auto-fill, minmax(300px, 1fr))` для автоматического расчета колонок.
* **Производительный живой поиск:** Единый алгоритм одновременно обрабатывает текст и параметры активного таба. Интегрирован паттерн `Debounce (250мс)` для защиты от избыточных перерисовок DOM, а также валидация на ввод пустых пробелов.
* **Стабильная пагинация без сдвигов макета:** При нажатии на «Load more» скрипт использует `DocumentFragment` и метод `appendChild`, мягко добавляя по 6 карточек в конец сетки. Ранее загруженные курсы не перезаписываются и не мигают.
* **Динамическая генерация интерфейса:** Кнопки категорий и счетчики количества курсов полностью рендерятся на основе актуального массива данных из `data.js`.
* **Доступность (Accessibility):** Тег ссылки `<a>` оборачивает только заголовок курса, а область клика расширяется на всю карточку через псевдоэлемент `::before`. Это сохраняет читаемость кода для скринридеров и естественный фокус с клавиатуры.

## Как запустить локально
1. Склонируйте этот репозиторий в локальную папку.
2. Откройте файл `index.html` с помощью расширения Live Server или напрямую в любом современном браузере.
