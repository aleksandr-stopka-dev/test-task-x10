# Тестовое задание для Команды 10X

### Стек: 
Vanilla JS, HTML5 (Семантика), SCSS (БЭМ методология), Сборщик (Vite/Gulp - укажи свой, если использовал).

### Что сделано на "боевом" уровне:
1. **Производительность:** Для живого поиска внедрен паттерн `Debounce`. Это предотвращает лишние перерисовки DOM при частых нажатиях клавиш.
2. **Архитектура CSS:** Стили разбиты на независимые БЭМ-компоненты. Архитектура легко масштабируется.
3. **Резиновый адаптив:** Вместо десятков жестких `@media` брейкпоинтов использована адаптивная сетка CSS Grid (`auto-fill`) и функция `clamp()`, обеспечивающая идеальную плавность от 1920px до 320px.
